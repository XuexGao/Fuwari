# Formally begin

> [! WARNING]
> 
> The Netlify OSS program does not accept personal blogs.

Make sure you have an open-source project that is hosted on Netlify, and your website link is also available on GitHub.
![](../../assets/images/2025-08-09-12-30-32-image.webp)

Go to https://opensource-form.netlify.com/

Fill in the relevant information and submit.

I wrote it like this

```bash
Open Source Team Application
Applicant Name: no name provided
Applicant Email: acofork@gmail.com
Project Name: afoim/fuwari
Proof of Contributors:
Code of Conduct URL: https://2x.nz
Project License URL: MIT license
Netlify Site URL: https://2x.nz/posts/static-web/#netlify
Notes:
```

# What is this?

It can connect to your VPS (provided CPU ≥ 1c, RAM ≥ 2G). Then run services on your machine, including

# Formally begin

Enter [My Servers - Zeabur](https://zeabur.com/servers)

Click to create

![](../../assets/images/2025-09-03-06-55-36-image.webp)

Click to add your own server

![](../../assets/images/2025-09-03-06-55-47-image.webp)

Reading requirements, continue

![](../../assets/images/2025-09-03-06-56-01-image.webp)

Fill in the SSH connection information to allow Zeabur to connect to your server.

![](../../assets/images/2025-09-03-06-56-30-image.webp)

Then, tools like k3s will be installed, and we will no longer need to manually SSH into the server—configuration can be done directly through the Zeabur web interface!

![](../../assets/images/2025-09-03-06-57-08-image.webp)

Next, we can try deploying the service; Zeabur deploys services to your self-hosted server for free.

![](../../assets/images/2025-09-03-06-57-41-image.webp)

![](../../assets/images/2025-09-03-06-57-52-image.webp)

Feel free to let your imagination run wild!
